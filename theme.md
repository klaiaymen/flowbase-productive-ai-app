# Flowbase UI Theme

## Design Direction

Flowbase should feel like a fresh, cozy workspace for structured thinking and visual collaboration. The product combines the calm document feel of Notion with the playful spatial energy of Miro, so interfaces should be clean, bright, lightweight, and confidently colorful.

## Color Palette

- App background: bright off-white with visible mint, cyan, violet, and soft yellow accents.
- Primary text: deep slate for strong contrast without feeling harsh.
- Surfaces: translucent white cards with subtle borders and soft shadows.
- Accent colors:
  - Mint and emerald for workspace health, creation, and calm progress.
  - Sky and cyan for AI, search, and system intelligence.
  - Amber and orange for tasks, calendar, urgency, and planning.
  - Violet and fuchsia for templates, spaces, and generative features.
  - Rose for notes, writing, and personal capture.
- Avoid single-hue themes, heavy beige palettes, muddy neutrals, and dark default dashboard styling.

## Typography

- Use a modern system sans stack: Inter first, then platform UI fonts.
- Headings should be confident but compact.
- Sidebar labels and metadata should use small, high-contrast text with generous line height.
- Avoid negative letter spacing. Use uppercase labels sparingly for navigation groups.

## Spacing And Shape

- Use dense but breathable spacing for productivity surfaces.
- Sidebar nav items should stay compact: around 32px tall.
- Cards and panels can use 20px to 28px radii for the cozy workspace feel.
- Buttons and controls should stay closer to 12px radii so they remain crisp and functional.
- Shadows should be soft and low-opacity, used to separate layers rather than decorate.

## Sidebar Guidelines

- Expanded sidebar width: about 224px.
- Collapsed sidebar width: about 68px.
- Expanded state shows logo, app name, group labels, menu text, and footer details.
- Collapsed state shows logo and colorful icon-only navigation with accessible labels.
- Every menu item should use a Lucide icon with a purposeful accent color and a subtle tinted icon tile.
- Group labels should be short, pill-like, muted, and scannable.

## UI Tone

- Keep dashboards useful on first load: show current work, quick actions, tasks, AI drafts, notes, calendar rhythm, and whiteboard activity.
- Prefer real controls and work surfaces over marketing copy.
- Use color as wayfinding, not decoration.
- Keep empty and starter states optimistic, concise, and actionable.
